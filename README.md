Em Java
	- As referências são usadas para identificar os objetos de forma excluisiva
	- Igualdade Referencial: compara a identidade de dois objetos
		* x == y
	Obs: Identidade não muda ao passar do tempo
	
	- Igualdade se refere a dois objetos serem iguais, não significa que são o mesmo objeto
	- Igualdade Estrutural:
		* e.equals(y)
	Obs: Igualdade muda ao passar do tempo
